## Steem-CMS-API

> 高仿Steemit，使用Larave5.5 + Vue2重写中心化部分。    
这是Vue2重写的前端部分。


#### 1. Language and framework  
ES6  
VueJS: 2.5.9  
[Bootstrap: 3.3.7-litera](https://bootswatch.com/litera)  
[vee-validate](http://vee-validate.logaretm.com)  

#### 2. Install and run  
Go in project path :
```bash
$ cd [project path]

$ npm install

$ npm run dev
```
